# Integer Behaviors



## Integer Division



### C++

```c++
#include <iostream>

int main() {
	int x = 8;
	int y = 5;
	std::cout << x / y;
	return 0;
}
```

Output : 

```
1 (integer)
```



### Go

```go
package main

import "fmt"

func main() {
	var result int
	result = (8 / 5)
	fmt.Println(result)
}
```

Output : 

```
1 (integer)
```



### Javascript

